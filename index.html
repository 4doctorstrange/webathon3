INTRODUCTION:

Hey Coder,have you recently started coding and are dealing with bugs and errors or planning to start coding??

Well then you are at right place as in comming section we will be discussing some

tips that every budding coder must remember.

"EVERY EXPERT WAS ONCE A BEGINNER".

Let's get started:

1.You learn by doing.Never let an analysis paralysis prevent you from ever starting.

2.Programming isn't like studying for a test.Memorizing stuff doesn't matter all that much.

3.Letting problems pile up is poor strategy.

4.Learning by yourself is really tough.Talking about code with other aspiring developers is a better way to learn.

5.There's a right way to ask for help.Make sure to ask following things if you need help.

  • Overcommunicate details that you see.
  • Explain exactly what you think should be happening.
  • Explain exactly what is actually happening.
  • Explain why you think it should be working differntly.

6.Do not forget to celebrate small wins.

7.It's common to give up right before change is about to happen.
